# Public Boundary

This repository is designed to be public.

It should contain:

- public-safe portfolio pages
- sanitized case studies
- architecture diagrams without private runtime details
- CV PDF
- public FastAPI Gateway source
- CI, smoke scripts, Docker files, and deployment notes

It must not contain:

- phone number
- private job-application trackers
- application answer banks
- private TeachClaw/OpenClaw source code
- teacher/customer data
- live VPS IPs
- API tokens, bot tokens, SSH keys, or `.env` files
- SQLite runtime databases
- Python cache files or virtualenvs

The public repo exists to prove the AI deployment story without exposing the private operating system behind TeachClaw.
